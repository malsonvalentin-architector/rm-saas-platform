📁 ФАЙЛЫ В ЭТОЙ ПАПКЕ
═══════════════════════════════════════════════════════════════

🟢 ГЛАВНЫЕ ФАЙЛЫ (для пользователя):
───────────────────────────────────────────────────────────────

   📘 ПЕРВЫЙ_ЗАПУСК.txt          Подробная инструкция для новичков
   
   🔧 INSTALL.bat                Установка программы (1 раз)
   
   ▶️  START.bat                 Запуск программы (каждый раз)
   
   🛑 STOP.bat                   Остановка программы
   
   🌐 ОТКРЫТЬ_В_БРАУЗЕРЕ.bat     Открыть интерфейс в браузере
   
   🔐 RESET_PASSWORD.bat         Сброс пароля администратора
   
   🔍 ПРОВЕРКА_СИСТЕМЫ.bat       Проверка что всё установлено


🔵 ТЕХНИЧЕСКИЕ ФАЙЛЫ (не трогать):
───────────────────────────────────────────────────────────────

   manage.py                     Django менеджер
   db.sqlite3                    База данных
   req.txt                       Список библиотек
   
   Папки:
   - rm/                         Настройки Django
   - data/                       Основное приложение
   - home/                       Главная страница
   - teleg/                      Telegram уведомления


⚡ БЫСТРЫЙ СТАРТ
═══════════════════════════════════════════════════════════════

Если Python УЖЕ установлен:

   1. Двойной клик → INSTALL.bat (один раз)
   2. Двойной клик → START.bat (каждый запуск)
   3. Готово! Браузер откроется автоматически


Если Python НЕ установлен:

   1. Прочитайте ПЕРВЫЙ_ЗАПУСК.txt
   2. Установите Python с python.org
   3. Запустите INSTALL.bat
   4. Запустите START.bat
